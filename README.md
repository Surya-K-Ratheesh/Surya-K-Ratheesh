## 🚀 Flagship Project — KidneyInsight

> AI diagnostic system for renal tumour detection from CT scans

| Feature | Detail |
|---|---|
| 🎯 Accuracy | **92% classification** across 3 tumour classes |
| 🧬 Model | Multi-model ensemble — ResNet50 + CLAHE preprocessing |
| 🔍 Explainability | Grad-CAM diagnostic heatmaps via Streamlit |
| 🔐 Access Control | Role-based (radiologist / doctor / admin) |
| 📄 Output | Automated PDF diagnostic report generation |
| 📁 Data | Full DICOM medical imaging pipeline |

```
Raw CT Scan (DICOM) → CLAHE Preprocessing → ResNet50 Ensemble → Grad-CAM Heatmap → PDF Report + RBAC
```
